# React boilerplate for Frontend Development

```bash
npm run install
```

> install all dependencies

<br>

```bash
npm run dev
```

> run the application on development mode

<br>

```bash
npm test
```

> run jest test

<br>

```bash
npm run eslint
```

> to run eslint

<br>

```bash
npm run eslint:fix
```

> to fix linting

<br>

```bash
npm run stylelint
```

> to run style lint

<br>

```bash
npm run build
```

> to build static files
